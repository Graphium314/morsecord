# 通常読み上げ

テキストチャンネルに投稿された文章を、投稿者ごとの速度と音程でCWに変換して再生します。

## 使い方

1. `/cw-join ch:<ボイスチャンネル>` でBotを参加させます。
2. `/cw-join` を実行したテキストチャンネルへ、読み上げたい文章を送信します。
3. `/cw-leave` で終了します。

例:

```text
CQ CQ DE JA1ABC
```

先頭が半角セミコロン（`;`）のメッセージは読み上げません。Botを退出させずに会話したいときに利用できます。

```text
;このメッセージは読み上げない
```

## 速度と音程

設定はDiscordユーザーごとに保存され、次回以降も使われます。未設定時は20 WPM、800 Hzです。

### `/cw-speed`

```text
/cw-speed speed:20
```

自分が投稿した文章の速度をWPMで設定します。指定できる最小値は5 WPMです。

### `/cw-freq`

```text
/cw-freq freq:800
```

自分が投稿した文章の音程をHzで設定します。指定できる最小値は10 Hzです。

これらの設定は通常読み上げ専用です。レッスンの速度・音程は、レッスン開始コマンドの引数で指定します。

## 文字の扱い

- 英字は大文字・小文字を区別しません。
- ひらがなはカタカナに、全角英数字は半角相当へ正規化されます。
- 空白や未対応文字は無音の区切りとして扱われます。
- 対応する英数字、記号、和文文字は[対応文字と音声仕様](対応文字と音声仕様.md)を参照してください。

## 反応しないとき

- Botがボイスチャンネルに参加しているか確認します。
- 投稿先が `/cw-join` を実行したチャンネルか確認します。
- レッスン中は通常読み上げが停止します。`/cw-end-lesson` で通常モードへ戻してください。
- メッセージが `;` で始まっていないか確認します。
