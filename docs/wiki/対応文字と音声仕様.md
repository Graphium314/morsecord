# 対応文字と音声仕様

## 対応文字

### 欧文

- 英字: `A`～`Z`
- 数字: `0`～`9`
- 記号: `! " $ & ' ( ) + , - . / : ; = ? @ _`

英字は大文字・小文字を区別しません。全角英数字も半角相当へ正規化されます。

### 和文

- カタカナ: `アイウエオ`から`ワヰヱヲン`まで
- 濁点、半濁点
- 長音 `ー`
- 読点 `、`

ひらがなはカタカナに正規化されます。濁音・半濁音は、基本文字に濁点・半濁点を続ける和文モールスとして送信されます。

## 未対応文字

未対応文字はエラーにはならず、無音の区切りとして扱われます。絵文字や上記以外の記号を含む文章では、意図しない間が入ることがあります。

連続する空白・未対応文字は1つの区切りにまとめられます。

## タイミング

速度はPARIS基準のWPMで、短点長は次の式です。

```text
短点長（秒） = 1.2 / WPM
```

- 長点: 短点3個分
- 同一文字内の信号間隔: 短点1個分
- 文字間隔: 合計で短点3個分
- 語間隔: 合計で短点7個分

再生開始時には短い無音が入ります。クリックノイズを抑えるため、信号の立ち上がり・立ち下がりには約10 msのコサイン型エンベロープが適用されます。
