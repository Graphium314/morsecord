# 聞き取りレッスン

短い問題を正解者が現れるまで繰り返し再生するモードです。複数人で同時に解答でき、終了時に成績を表示します。

## 基本的な使い方

1. `/cw-join` でBotをボイスチャンネルへ参加させます。
2. `/cw-start-lesson` を実行します。
3. 聞き取った内容を、操作用テキストチャンネルへ投稿します。
4. 終了時に `/cw-end-lesson` を実行します。

引数を省略した開始例:

```text
/cw-start-lesson
```

既定では、日本のコールサインを15～20 WPM、500～1000 Hzから問題ごとにランダムな設定で出題します。

## 開始時の引数

```text
/cw-start-lesson min_speed:<WPM> max_speed:<WPM> min_freq:<Hz> max_freq:<Hz> probset:<問題セット>
```

| 引数 | 内容 | 既定値 | Discord上の最小値 |
| --- | --- | --- | --- |
| `min_speed` | 速度範囲の下限 | 15 WPM | 5 |
| `max_speed` | 速度範囲の上限 | 20 WPM | 5 |
| `min_freq` | 音程範囲の下限 | 500 Hz | 200 |
| `max_freq` | 音程範囲の上限 | 1000 Hz | 200 |
| `probset` | 問題セット名 | `call_ja` | ― |

下限は上限以下である必要があります。同じ値を指定すると速度・音程を固定できます。

```text
/cw-start-lesson min_speed:20 max_speed:20 min_freq:700 max_freq:700 probset:rand5_jp
```

片方の境界だけを指定した場合は、指定値を含むように既定の境界が調整されます。意図を明確にするため、範囲を変更するときは上下限を両方指定することを推奨します。

## 解答と進行

- 不正解: ❌
- その問題で最初の通常解答による正解: 🥇
- 2人目以降の通常解答による正解: ⭕

問題は「問題の再生時間＋10秒」おきに繰り返されます。最初の通常解答による正解から5秒後に次の問題へ進むため、その間はほかの参加者も解答できます。

英字の大文字・小文字は区別しません。`rand5_jp` では、ひらがな・カタカナや全角・半角なども正規化して判定します。

### 伏字で解答する

Discordの伏字記法で解答を囲むと、ほかの参加者に答えを見せずに判定できます。

```text
||JA1ABC||
```

伏字の正解には⭕が付きますが、問題は次へ進まず、成績にも加算されません。経験者が初心者と同じ問題に参加するときなどに利用できます。

## 問題セット

| `probset` | 出題内容 | 解答例 |
| --- | --- | --- |
| `call_ja` | 日本のコールサイン。まれに移動運用の `/数字` を付加 | `JA1ABC`、`7K1ABC/1` |
| `nr_allja` | ALL JAコンテスト形式の都府県支庁番号＋出力記号 | `13M`、`5NN 13M` |
| `nr_acag` | 全市全郡コンテスト形式の市・郡・区番号＋出力記号 | `100101H`、`5NN 100101H` |
| `rand5_jp` | 清音・濁音・半濁音を含むランダムなカタカナ5文字 | `イロハガパ` |
| `file:<ファイル名>` | Bot側の `lesson_txt` に置いたファイルからランダムに1行 | ファイルの内容による |

`nr_allja` と `nr_acag` の解答では、先頭の `5NN` は付けても省略しても正解になります。出力記号は `H`、`M`、`L`、`P` のいずれかです。

### 独自問題ファイル

Botを運用している環境の `lesson_txt/` にUTF-8テキストファイルを置き、1行に1問を記述します。空行も問題として読み込まれるため除いてください。ファイル名に `/` は使用できません。

```text
/cw-start-lesson probset:file:questions.txt
```

解答メッセージは大文字化して比較されるため、英字を含む問題は大文字で記述してください。また、音声化できる文字だけを使用してください。

## 終了と成績

`/cw-end-lesson` を実行すると通常読み上げへ戻り、次の情報を表示します。

- `total questions`: 再生された問題数（終了時の出題中の問題を含む）
- `average retry`: 1問あたりの平均再生回数
- `🥇 / ⭕`: ユーザーごとの最速正解数 / 通常解答での正解数

参加者は最速正解数の降順で表示されます。`/cw-leave` でもレッスンは終了しますが、成績を確認したい場合は先に `/cw-end-lesson` を実行してください。
