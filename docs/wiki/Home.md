# MorseCord Wiki

MorseCordは、DiscordのボイスチャンネルでCW（モールス）を再生し、聞き取り練習を行うBotです。

- チャットに送った文章をCWで読み上げる
- 日本のコールサイン、コンテストナンバー、和文などを繰り返し聞き取る
- ランダムな長文を一度だけ聞き、入力結果との違いを確認する

## まず使ってみる

1. Botを利用するテキストチャンネルで `/cw-join ch:<ボイスチャンネル>` を実行します。
2. Botが参加したら、同じテキストチャンネルへ `CQ CQ DE ...` などと送信します。
3. 終了するときは `/cw-leave` を実行します。

`/cw-join` を実行したチャンネルが操作用テキストチャンネルになります。通常読み上げの投稿とレッスンの解答は、このチャンネルで行ってください。ボイスチャンネル内のテキストチャットも指定元として利用できます。

## コマンド早見表

| コマンド | 用途 |
| --- | --- |
| `/cw-join ch:<チャンネル>` | 指定したボイスチャンネルへ参加 |
| `/cw-leave` | ボイスチャンネルから退出 |
| `/cw-speed speed:<WPM>` | 自分が送った文章の読み上げ速度を変更 |
| `/cw-freq freq:<Hz>` | 自分が送った文章の音程を変更 |
| `/cw-start-lesson` | 繰り返し再生する聞き取りレッスンを開始 |
| `/cw-end-lesson` | レッスンを終了して通常読み上げへ戻る |
| `/cw-start-long-lesson` | 一度だけ再生する長文練習を開始 |
| `/neko [count]` | `にゃーん`と返答（1～32回、既定値1） |

コマンドの詳細は次のページを参照してください。

## 利用時の注意

- コマンドはサーバー内でのみ利用できます。DMには対応していません。
- 1つのサーバーにつき、同時に参加できるボイスチャンネルは1つです。
- レッスンを切り替えるときは、先に `/cw-end-lesson` を実行してください。
- Botが反応しない場合は、Botがボイスチャンネルに参加済みか、投稿先が `/cw-join` を実行したチャンネルかを確認してください。
- Bot管理者向けの起動方法や必要権限は[Botの導入と運用](Botの導入と運用.md)にまとめています。
